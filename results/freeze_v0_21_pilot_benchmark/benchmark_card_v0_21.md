# OncoGuard-Response v0.21 Pilot Benchmark Card

## Benchmark name

OncoGuard-Response v0.21 Pilot Benchmark

## Purpose

OncoGuard-Response is a proof-of-concept multi-visit oncology therapeutic-response authorization benchmark. It evaluates whether AI systems authorize therapy continuation, defer or escalate evaluation, hold therapy, switch therapy, or trigger emergency toxicity management across longitudinal oncology evidence states.

The benchmark is designed to test therapeutic-response governance behavior rather than single-turn treatment recommendation accuracy.

## Scope

This v0.21 release is a frozen pilot benchmark intended for methods validation, policy simulation, and initial model evaluation.

It is not intended to estimate real-world oncology AI safety rates or represent the full complexity of oncology practice.

## Composition

The benchmark contains:

- 20 oncology response trajectories
- 3 visits per trajectory
- 60 visit-level therapeutic authorization decisions

Each trajectory includes:

- trajectory ID
- title
- action path
- failure being tested
- authorization dependency

Each visit-level record includes:

- trajectory ID
- visit number
- expected action
- inferred evidence state
- stage-adjusted evidence status
- failure being tested
- authorization dependency

## Allowed authorization actions

The benchmark uses five authorization action labels:

- `continue_therapy`
- `hold_therapy`
- `switch_therapy`
- `escalate_evaluation`
- `emergency_toxicity_management`

## Failure and evidence-state concepts

The pilot includes therapeutic-response failure modes involving:

- missing critical evidence
- incomplete toxicity assessment
- contradictory evidence
- benefit overauthorization risk
- other authorization dependencies

The v0.21 analysis introduced `evidence_status_stage_adjusted` to avoid incorrectly applying trajectory-level evidence labels to all visits within a trajectory. This distinguishes the overall failure mode being tested from the visit-level authorization state.

## Metrics

Primary metrics include:

- PTMAR: premature therapy misuse action rate
- AACR: authorization alignment contract rate
- defer rate
- unsafe authorization rate
- over-deferral rate
- correct authorization rate

## Labeling and construction pipeline

The benchmark was authored as a synthetic oncology therapeutic-response authorization dataset. Trajectories were designed to test whether therapy continuation is authorized by the current evidence state across sequential visits.

The v0.21 visit-level dataset was generated by expanding trajectory-level action paths into 60 visit-level decision records. Stage-adjusted authorization status was added after auditing conflicts caused by applying trajectory-level evidence labels uniformly across visits.

## Known limitations

This is a pilot benchmark with important limitations:

- synthetic trajectories
- modest sample size
- 20 trajectories / 60 visit-level decisions
- no structured raw laboratory values or imaging reports yet
- no external oncology expert adjudication yet
- limited treatment-class diversity
- correlated visits within each trajectory
- not intended to estimate real-world oncology model safety rates

## Intended use

This benchmark should be used for:

- proof-of-concept model evaluation
- therapeutic authorization policy simulation
- failure taxonomy development
- comparison of model behavior against explicit authorization contracts
- future expansion into larger multi-visit oncology benchmarks

## Future directions

Planned extensions include:

- expansion to 50–100+ trajectories
- multi-model evaluation
- external oncology expert review
- real-world or semi-synthetic EHR-style oncology trajectories
- richer treatment classes
- structured lab, imaging, toxicity, and response evidence
- stronger confidence intervals and subgroup analyses

## Recommended citation language

OncoGuard-Response v0.21 is a pilot multi-visit oncology therapeutic-response authorization benchmark designed to evaluate whether AI systems authorize therapy continuation only when supported by the current evidence state.
